Para fazer com que apareça o Emmit do nunjunck

é preciso instalar um plugin no VS code

o nunjunck template e em seguida realizar as seguintes configurações:

  • execulte "ctrl" + "shift" + "p" e digite: "settings:json"

  • "files.associations": {"*.html":"njk"},
  • "emmet.includeLanguages": { "njk": "html"},
  • "vsicons.associations.files": [ { "icon" : "nunjuncks", "extensions": ["njk"], "format": "svg" }
  • "workbanch.iconTheme" : "vscode-icons"
